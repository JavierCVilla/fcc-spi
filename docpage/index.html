About FCCSW

FCCSW is the software for the FCC experiments. We aim to support all detector concepts for all accelerator options (electron-electron, electron-hadron and hadron-hadron). Therefore, we try to create software that is independent of detector or experiment specifics.

This page collects release information and documentation. Check individual releases for doxygen documentation and links to the releases on github.
